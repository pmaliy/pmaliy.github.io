7. Формы

Определили понятие форм в HTML, обсудили их назначение и возможности. Поговорили о методах отправки форм и значении атрибута name элементов формы.

Познакомились со следующими тегами:

  • form - родительский элемент формы, обязательные атрибуты action и method
  • input - элемент для ввода данных. В зависимости от атрибута type может отображаться как текстовое поле, поле для ввода пароля, флажок (checkbox), переключатель (radio) или кнопка

  • label - inline-тег, который можно привязать к тегу input используя атрибут for, которому нужно для этого задать значение, равное значению атрибута id соответствующего input-a
